I want to develop a circuit simulation plugin for siyuan-note which can be embedded inside a note using slash commands and circuitjs1. circuitjs1 is a jasvascript library/project allowing for circuit simulation on the browser. 

## Subfolders:

1. `circuitjs1`: the source of circuitjs1 whose readme.md contains info on how to build it and embed it inside other applications and webpages. Also use it as a reference for figuring out how to interact with it.
2. `siyuan-plugin-api-reference`: the siyuan plugin api for reference
3. `plugin-sample`: sample siyuan plugin for reference

Look into `circuitjs1` build and integration instructions to figure out how to do that and let me now if you need additional tooling/tools to build it. Eventually from what I understand you'd have a circuitjs.html which can be embedded as a block inside siyuan. Good luck !
